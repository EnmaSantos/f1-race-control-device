# Web Dashboard

The dashboard is planned for Phase 4.

Initial goals:

- Show current replay event.
- Show recent race-control messages.
- Show session status.
- Show favorite driver status.
- Add manual test buttons for hardware events.

Suggested stack later:

- React
- Vite or Next.js
- Local backend API from `backend/`

Do not build this before the hardware alert box and replay backend are working.
